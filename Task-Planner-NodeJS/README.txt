This program runs on port: 9999 but can be changed from the top of the program

While on port 9999, link is:
http://localhost:9999/ 

Provided JSON file contains an example task but is not necessary to run program.
It will automatically create a new one if one cannot be found.

Warning: If you create your own taskRecords.json the file *must* contain "[]" to showcase an empty JSON file. 
"[]" is created automatically if you allow the program to create it's own taskRecords.json which is recommended.
